Crisis Communicator

Developed a device that when switched on, sends vitals information of wearer to an end device and communicates an emergency signal via SMS.
Onboard sensors monitor heart rate, skin temperature and motion detection connected to an XBee radio module that sends the information to another coordinator XBee device. The coordinator device connects to a Raspberry Pi base station that uses multi-thread Python code to process the vitals data and send emergency alerts via SMS and PushBullet.

Embedded Intruder Detection System

Used an STM32 development board, a PIR sensor to detect intruder motion and an I2C camera module to take a picture of an intruder when motion is detected. The system was developed using the FreeRTOS platform and 4 real-time tasks were made with different priorities, an interrupt handle and GPIO was configured to interface the PIR sensor and camera.
